# Project Structure

This repo is organized into a few main areas:

- `src/` - React app code
- `server/` - Node server and API code
- `extension/` - Chrome extension files
- `public/` - static files used by the website
- `scripts/` - helper scripts and local build utilities
- `docs/` - notes and deployment docs
- `design/` - branding files, fonts, mockups, and other reference assets
- `prototypes/` - old HTML experiments kept for reference

Most day-to-day work should stay inside `src/`, `server/`, `extension/`, and `public/`.
